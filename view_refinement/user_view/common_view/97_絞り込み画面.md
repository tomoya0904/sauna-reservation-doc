### 1.2 要件

- 1~27の項目はAND検索とする
- 各項目内での検索はOR検索とする

1. **料金設定**
- 0 ~ 60,000円で選択可能
  - 0 ~ 10,000円は 1000円刻み
  - 10,000 ~ 60,000円は　5000円刻み
- 下限値の入力のみの場合は ◯円以上、上限値のみの入力の場合は◯円以下を指す
- 下限値が上限値を上回るものは設定不可 (10,000円 ~ 1,000円　はできない)
